# Mohammed Ali — QA Automation Engineer (Portfolio)

This is a Vite + React single-page portfolio website with Tailwind CSS and Framer Motion.

## Quick local setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run dev server:
   ```bash
   npm run dev
   ```
   Open http://localhost:5173

3. Build:
   ```bash
   npm run build
   ```

## Deploy to Vercel

1. Push this repo to GitHub.
2. Sign in to https://vercel.com and import the GitHub repository.
3. Vercel should autodetect the Vite project. Set build command `npm run build` and output directory `dist`.
4. Deploy — Vercel will provide a live URL.

You can edit contact details in `src/App.jsx`.
